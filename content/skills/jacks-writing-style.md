## Why it exists

Every public surface under my name was drifting into assistant-voice: hedged, bland, "it's worth noting." This skill is the fix — examples from real READMEs, blogs, and changelogs.

## History

### 1.0.0
First packaged for AutoVault with reference corpora (blog, drunk.support, README examples) and a Claude-ism detector script.

## How Jack actually uses it

Load it whenever prose goes out under Jack's name — even if nobody says "in my voice." Default for first-person technical writing unless told otherwise.

## What it is not

Not a generic tutorial. It's the house runbook for this machine and these agents.
