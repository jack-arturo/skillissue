Install

AutoVault · Skills CLI

AutoVault (preferred)

Skills live in a local vault, get validated, then sync into each agent’s normal skill directory as filesystem links. No MCP required for day-to-day use. 

autovault add jack-arturo/skillissue --sync-profiles

Skills CLI

Search the open ecosystem, then install a package when you already know the slug. 

npx skills find
npx skills add <package>

About

This site is a public window onto skills Jack actually runs — writing voice, Cloudflare ops, browser automation, PR babysitting, inbox triage, desktop control, and the meta-skills that keep the vault honest.

The domain is a buried pun. skill issue is a gamer insult; .sh is shell. If your agent can’t land a change, maybe it needs a better skill — not a longer prompt.

Source of truth for installable packages is AutoVault / the Skills CLI. This site catalogs the collection and ships as a static Cloudflare Pages project with no build step.
