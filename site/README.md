# skillissue.sh — site

Static skills hub. No build step. Cloudflare Pages hosts it.

## Files

- `index.html` — the page (inline CSS + tiny JS for filter/copy)
- `skills.json` — catalog consumed by the page and agents
- `llms.txt` — agent-oriented summary
- `_headers` — security + cache headers
- `assets/` — favicon + OG image

## Deploy

Push to `main`. See root README.
